CSS3 Transformations

Two-Dimensional (2D) Transformations

Circle

CSS3 transformations are very similar to mathematical transformations; they allow you to alter a shape’s position, size, dimension or orientation. In CSS, transformation is a new type of property that can be applied to almost any standard HTML element. The logo on the right is a live example of this. It is transformed from a circle to a kite using standard CSS styles. Hovering over the logo will revert it back to its original shape. 3D transformations are not within the scope of this tutorial since it requires understanding of 3D shapes which is taught in A-level Further Maths.


CSS3 transformations consist of 4 types:






  • Translation
  • Rotation
  • Scale/Enlargement
  • Skew
CSS3 Transformations

Translation

This type of transformation allows a user to change the position of a HTML element. The translation property value is commonly used when defining a slide-in animation. The main header of this page makes use of translation. The type of transformation is defined in the property’s value section of a CSS code block. To understand any transformation you will need to recap a bit of GCSE maths. A translation property moves an object Npx in the ‘horizontal-direction’ (along the x-axis) and Mpx in the ‘vertical-direction’ (along the y-axis). Hence the code is:



#Box:active { …  transform: translate(Mpx, Npx); …}

/*Replace “…” with default properties such as size, default location, absolute position etc.*/

The example below is translated 200px to the right and 100px above (hence -100px) using the ease timing function. Notice the use of the 'active' CSS state for this example. To see the effect, you will need to click and hold the left mouse button.

CSS3 Transformations

Rotation

Rotations are one of the simplest form of transformation. A good analogy of CSS3 rotations is the rotating of car wheels when they are moving. CSS3 rotations can be used as a gimmick or to reveal information behind an element. The main logo on each of this product uses this transformation type to perform a ‘flip’ effect. To define a rotation you only need specify by how much amount to rotate an object in degrees. The symbol for degrees in CSS is just ‘deg’! By default, positive values correspond to clockwise rotation and negative values correspond to anticlockwise rotation.



#Box:active { …  transform: rotate(405deg); …}

/*Replace “…” with default properties such as size, default location, absolute position etc.*/

The example below is rotated 405° clockwise using the linear timing function. To see the effect, you will need to click and hold the left mouse button on the box below.

Scale (Enlargement) & Stretch

The scale transformation allows users to enlarge or reduce an object’s height, base or both dimensions. Scale can be used to create interactive buttons or logos which expand on hover. The scale value can be defined from 0 to N. A ‘0.5’ scale value for example, is ¼ of the original shape’s area as ½ * ½ = ¼. Beyond 1, the scale is used for enlargement e.g. a ‘2’ scale value means double the length & width. ScaleX or ScaleY can be used to define a scale in only 1-dimension making it effectively a stretch



#Box:active { … transform: scale(1.5); …}

/*Replace “…” with default properties such as size, default location, absolute position etc.*/

Skew

This is one of the most complex form of transformations. Its use must be carefully evaluated as it generally cannot be used for just the sake of it. A skew is a type of transformation where the one or more of the shape’s dimensions is adjusted without affecting the overall area of the shape. Hence for example if the width of a shape is reduced its height increases and vice versa. Click the image below to see an example of a skew in the x-direction.


A CSS3 skew transformation is defined in the same way as translate: using angles (instead of pixels) along the x-axis and the y-axis. To get the desired skew you will often need to test and experiment with multiple values. The source code is shown below and some examples are displayed on the next page.



#Box:active { … transform: skew(20deg, 40deg); …}

/*Replace “…” with default properties such as size, default location, absolute position etc.*/
