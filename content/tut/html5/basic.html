Basics of HTML

What is HTML?

HTML (Hyper Text Transfer Protocol) is the official language that is used to display content in the form of various elements on a webpage. The format used by HTML differs from most traditional programming language as it uses 'tags' to define a type of elements e.g. "<DIV />" rather than variables e.g. "Dim Rect As ..."


The first HTML document was created in 1990 by Tim Berners-Lee founder of the world wide web. The primary purpose and advantage of HTML from its predecessors is that it allows inter-linking of webpages using 'hyperlinks'.


An HTML document can include one or more of the following:

  • Plain Text
  • Images
  • Audio
  • Video
  • Scripts

Basics of HTML

Tags

All HTML elements and the syntax itself follow an XML format. This means that they are written using tags. An open HTML tag is defined by the less-than symbol (<), the tag’s name (div) and the more-than symbol (>). In order for the browser to correctly detect and display the tag, it must be closed. This can be done in the same way as opening it with the exception of adding a forward slash (‘/’).
A sample tag will look like: <DIV></DIV>.
All content will then come between the two tags: <DIV>Content Here…</DIV>.
Within the open (first) tag, several properties and their corresponding values can be set. A property is added after a space and its value is defined by =“VALUE”.


Below is an example of a tag (LINK) whererel is one of its properties and stylesheet is the property's value which is used to inform the broswer that the file to be linked is a stylesheet. 



  
<link rel="stylesheet" type="text/css" href="css/style.css"/>

Structure

It is important to understand the general structure of an HTML document. The structure is an industry standard and is used by website designers worldwide. The list below illustrates this structure. An HTML document is just a collection of links and plain text in its simplest form. This means that contrary to most editing software available today (e.g. Word Processor or Visual Studio), an HTML document can only embed text inside it. All multimedia is linked to an HTML document

Most websites today follow this order:

  • HEAD
    • META
    • TITLE
    • STYLESHEETS

     

  • BODY
    • SCRIPT
    • HEADER
    • NAV
    • SECTION/ARTICLE/ASIDE
    • FOOTER
Basics of HTML

Head Tag

Most content that is not directly visible to the user goes into the head tag. The head tag is like a metadata of a picture holding information such as image size, location, date taken etc. In HTML this is mainly used for linking Meta, Title, Favicon and Style-sheets. Hover over the highlighted text to see a preview image where applicable.


Below is the code of a sample head tag:



<head>

  <meta http-equiv="Content-Type" content="text/html;  charset=utf-8" />

  <title>Sample Title</title>

  <link rel="shortcut icon" type="image/png"  href="img/style/favico.png">

  <link rel="stylesheet" type="text/css"  href="css/style.css"/>

  </head>

Body Tag

All visible content is placed in the body tag of an HTML document. Since the body and head tags are both primary tags, there can only be one instance of them in a single HTML document. Sub-Tags of Body Explained


  1. Script – This is where a developer can either link or directly code JavaScript snippets. They allow a web developer to extend the functionality of their website dynamically with the help of coding in C-syntax.
  2. Header – A new tag which is commonly used by HTML5 websites today. This specifies content that must be displayed and anchored on the top of the page e.g. the title bar above.
  3. NAV – Web designers can specify a top or side (like this product) menu bar in this section
  4. Section/Article/Aside – These tags can interchange and you may only find only 1-2 of them even in modern websites. This is where the main text is recommended to be placed in.
  5. Footer – Like the header the footer is anchored on the bottom of the page. The footer is a place where developers normally place copyright information and version info of their site.
