In this section you will be familiarised by some other HTML features that are used to enhance a user’s experience when visiting a webpage. This section will only be limited to the HTML portion of a webpage.

Title

A Title simply put is the user-friendly name of the webpage which is displayed on the front-end to the users. Most modern browsers display the title name on each individual tab opened in a browser. This way users can know which tab contains what. <<ADD PIC SHOWING TITLE? Or 2 in 1 later?>>
The syntax of the Title tag needs no explanation except for an important note that it must be added in the head tag. Basically, a pair of ‘title’ tags is required and then just like the P or H1 tag, text is added within the tags.



<title>HTML5 Tutorials</title>
Other

Favicon

A favicon is a small graphics icon that acts as a logo of the website being currently accessed. In simple terms the icon is of *.ico or *.png format and generally very small sized e.g. 16x16 pixels. This icon is then displayed on the browser tab next to the webpage’s title. This can be used to identify easily which tab contains the opened website as then the user is not required to manually read the titles of all webpages.
<<SCRSHOT OF LOC. OF FAVICO IN A BROWSER>>
The syntax is just one-line but it is very important to remember that the favicon can only be added in the head tag of a page and commonly placed right below the title tag.


<link  rel="shortcut icon" type="image/png"  href="../../img/style/favico.png">
Other

Syntax of Favicon Explained


  • Link – The name of the tag. This basically is used whenever some information is needed to be applied or integrated to a webpage from an external source. A common usage of a link tag is to apply styles using external stylesheets.
  • Rel – Defines the category of content is being linked. The ‘shortcut icon’ tells the browser to display the relevant image provided in the favico area as well as when pinning the webpage to desktop and/or the taskbar.
  • Type – The format of data being supplied. It is always important to double check the format of the original file and the type value is the same by looking at the end of the filename after the (last) period.
  • Href – The actual absolute or relative link of the file. This can also from the web if accessible.

Tooltips

Tooltips is used to display more or underlying information when a user hovers over an object. Simple tooltips don’t require any javascript knowledge and can be added to any element. You may have noticed use of tooltips when you hover over some top menu bar items. <<ADD IMG OF TOOLTIP>>
The tooltip can be added by including the ‘title’ property (not tag!) of an element
Syntax: 

<a href="#basic" title="Basics of  HTML" target=“_blank”>Basics</a>
