The first picture (on the right) starting from the top illustrates the various components found on the homepage. To zoom-in an image, simply click anywhere on the image except for the description bar which reads 'Click to zoom'. Once zoomed in the description bar will contain description of the image being displayed. To close an image after viewing it, simply hover over the top-right corner of the image and a 'X' button will appear which can be clicked to close the image.

The second picture illustrates the common controls found on most of the pages of the EBook. The top menu bar is the default method of navigation thorughout the website. It can be used to select individual 'topics'. A topic will typically consist of 4 'sub-topics' including a summary. To goto the next sub-topic, simply click on the required sub-topic name from the side bar which is docked on the left of each topic page (not visible in this page). Finally to move forth and back between pages of the same sub-topic simply use the 'Next' and 'Prev' buttons respectively. Alternatively, the keyboard arrow keys can also be used for navigation.

The last picture shows the controls for a video embed in the EBook. These controls may vary depending upon the broswer. To learn how to play 'JPong' use the in-game help function. Finally, to learn about the aspects and purpose of these elements and to know more about them in depth, view the Evaluation PDF which is located in 'More > Evaluation'. To return to what you were reading previously (except homepage), click the '← Go Back' button.
